# koa-server



## pnpm

建议 node >= 18.12.0

安装pnpm

```
npm install -g pnpm@latest-10
```

和 npm 命令的对比

| npm 命令        | pnpm 等效                                       |
| --------------- | ----------------------------------------------- |
| `npm install`   | [pnpm install](https://www.pnpm.cn/cli/install) |
| `npm i <pkg>`   | [pnpm add ](https://www.pnpm.cn/cli/add)        |
| `npm run <cmd>` | [pnpm ](https://www.pnpm.cn/cli/run)            |
